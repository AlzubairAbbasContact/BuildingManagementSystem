# 🏢 Building Management System (BMS)

A web-based application built using PHP and MySQL to manage apartments, tenants, and payments in an organized and efficient way.

---

## 🚀 Features

- Manage apartments (vacant / occupied)
- Add and manage tenants
- Track payments with dynamic remaining balance calculation
- Prevent overpayment automatically
- Cancel payments with reason tracking
- Soft-delete system to preserve data history
- Simple dashboard for system overview

---

## 🛠️ Technologies Used

- PHP
- MySQL
- HTML
- CSS
- MVC Architecture

## 📌 Project Overview

This project was developed to simulate a real-world building management system and improve backend development skills using PHP and MySQL.

It focuses on implementing business logic such as payment validation, tenant management, and maintaining data integrity through structured design.

---

## ⚙️ How to Run

1. Clone the repository
2. Import the database (`bms_db.sql`)
3. Run migration files
4. Configure database connection in `config.php`
5. Start your local server and open the project
